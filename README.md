# DognGo

## Installation

> Clone or download the repository and install the dependencies

```bash
# Clone the repository
git clone --depth 1 https://github.com/berenicebou/dogngo.git

# Go to dogngo directory
cd dogngo

# Install dependecies
npm install

# Start the project with Expo
expo start
```
